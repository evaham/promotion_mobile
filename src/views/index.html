본부시스템 안내 FAQ

투게더 POS 본부시스템이란?

  • 1 다점포 매장용 관리 솔루션

  • 2 설치가 필요 없는 웹 기반 솔루션으로 어디서든 접근 가능 (크롬, 엣지 권장)

  • 3매장 운영상황에 맞게 2가지 버전 중 채택 가능

    미통합 버전

    매장 DB 통합 없이 기존대로 사용 가능

    매입, 매출을 한눈에 조회할 수 있는 화면 제공

    통합버전

    전체매장 DB 통합(상품, 분류, 공급사) 필요

    전체매장의 상품, 공급사, 분류 데이터를 통합하여 조회, 관리할 수 있다.

투게더 POS 본부시스템 신청 절차

  • 1본부시스템 통합/미통합 버전 확정

  • 2도입 희망일 선택
    통합 여부에 따라 매장 상품장 데이터 검수 필요

  • 3접속 도메인(URL) 결정

  • 4도메인 생성, 본부DB 구성, 웹페이지 배포

  • 5데이터 검수 완료 후 매장 데이터 적용 일자 확정 및 파트너사에서 로컬서버 셋팅

  • 6각 매장 데이터의 정상적 수집 확인

  • 7본부그룹에 매장이 추가되는 경우 각 매장의 파일자료실 본부DB로 로컬서버 셋팅

투게더 POS 본부시스템 비용

DB 검수 및 도입비매장당 10 만원

월 사용료OPR 카피당 2만원

  • 1. 본부시스템 도입 시 이전 데이터를 이어서 볼 수 있나요?

    ✔통합버전은 모든 매장들이 상품, 분류, 공급사를 통합하여 사용하기 때문에 기존 DB와 분리가 필요합니다.
    따라서 기존 데이터를 이어서 볼 수 없으며, 별도 MGR에서 이전데이터를 볼 수 있습니다.

    단, 기준 매장의 DB로 본부DB를 구성할 경우 기준 매장은 데이터를 이어서 볼 수 있습니다.

  • 2. 기존 POS 프로그램과의 차이점은 무엇인가요?

    ✔매장에서 상품의 주요정보(상품명, 바코드, 분류, 과면세, 판매방식, 중량정보)를 수정할 수 없습니다.

    ✔매장에서 상품 등록(MGR 상품등록, 미등록상품 자동저장, GoodsUtil, OPR긴급등록)을 할 수 없습니다.

  • 3. 점간이동이 가능한가요?

    ✔본부사이트 '재고관리>점간이동' 메뉴에서 출고매장과 입고매장을 지정

    ✔ 이동할 상품을 선택하고 전표 작성

    ✔ 작성한 전표는 출고매장의 핸드터미널 수신리스트에 반품전표로 수신됩니다.

    ✔ 출고매장에서 전표처리가 완료되면 입고매장의 핸드터미널 수신리스트에 매입전표로 수신됩니다.

  • 4. 본부시스템에서 상품 일괄등록과 수정이 가능한가요?

    ✔ 기초관리>일괄관리 메뉴에서 일괄등록과 수정이 가능합니다.

  • 5. 본부시스템과 매장데이터는 실시간 동기화되나요?

    ✔ 5~15분 사이에 동기화됩니다
